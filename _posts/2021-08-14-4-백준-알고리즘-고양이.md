1. 10171번 10712
<br>10171번과 10172번 모두 printf에서 특수기호를 출력하는 방법을 알아야 풀 수 있는 문제였다.
<br><br> 
2. 문제 
10171번
>아래 예제와 같이 고양이를 출력하시오.


 10172번
 >아래 예제와 같이 개를 출력하시오.

3. 출력: 
>10171번
\    /\ 
 )  ( ') 
(  /  ) 
 \(__)| 

 >10172번
 |\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
3. 정답
10171번
```C
#include <stdio.h>

int main() 
{
	printf("\\    /\\\n");
	printf(" )  ( ')\n");
	printf("(  /  )\n");
	printf(" \\(__)|\n");
	
	return 0;
}
```
10172번
```c
#include <stdio.h>

int main(void) 
{
	printf("|\\_/|\n");
	printf("|q p|   /}\n");
	printf("( 0 )\"\"\"\\\n");
	printf("|\"^\"`    |\n");
	printf("||_/=\\\\__|\n");
	
	return 0;
}
```
4. printf 에서 특수기호 사용하기
```
\' : 작은따옴표
\" : 큰 따옴표
\? : 물음표
\\ : 백 슬래시(\)
\n : 줄 바꿈, 개행, new line
\t : 수평 탭 (tab)
```
